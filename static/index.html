Identifile

File Type Identifier

Do you have a file that you are unsure of its extension or type? Identifile was made just for that. It works by checking the file’s “magic number” — a unique signature stored in the file header — to accurately identify what the file really is.

This tool can be used to detect unknown file types, file extension spoofing (e.g an .exe disguised as .jpg) and embedded files (one file hidden within another).


🛡️
      Disclaimer:
      Files are analysed server-side but never stored.
      Only the first 256 KB of each file is read for analysis.
      No data is written to disk.
      As good practice, avoid uploading sensitive files to this tool or any other tool with similar functionalities.
